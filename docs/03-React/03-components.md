# Components

## ConnectDialog

A pre-styled dialog

```jsx
import { ConnectDialog } from "@connect2ic/react"

<ConnectDialog dark={false} />
```

### Props

#### dark


## ConnectButton

A pre-styled button for connecting & disconnecting

```jsx
import { ConnectButton } from "@connect2ic/react"

<ConnectButton 
  dark={false}
  onConnect={() => {/* connected */}}
  onDisconnect={() => {/* disconnected */}}
/>
```

### Props

#### dark

#### onConnect

#### onDisconnect
